Base

Core

Basic framework functionality.

Building blocks

Entity

Long-lived object defined by its identity.

Value object

Immutable object solely defined by its attribute values.

Aggregate

Set of objects bound by a root entity.

Factory

Object that create other domain objects.

Repository

Object that store and retrieve an aggregate.

Service

Standalone logic within the context of the domain.

Domain event

Capture the memory of something interesting which affects the domain.

Policy

Encapsulate business rules as interchangeable strategies.

Assembler

Maps between domain and interface objects.

Finder

Object that retrieves data efficiently.

Interfaces

Pagination

Assisted slicing of big data sets.

REST

Stateless Web resources.
